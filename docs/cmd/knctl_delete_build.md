## knctl delete build

### Synopsis

Delete build

```
knctl delete build [flags]
```

### Examples

```

  # Delete build 'build1' in namespace 'ns1'
  knctl delete build -b build1 -n ns1
```

### Options

```

  -h, --help               help for build
  -n, --namespace string   Specified namespace (or default from kubeconfig)
```
