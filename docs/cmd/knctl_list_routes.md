## knctl list routes

List routes

### Synopsis

List all routes in a namespace

```
knctl list routes [flags]
```

### Examples

```

  # List all routes in namespace 'ns1'
  knctl list routes -n ns1
```

### Options

```
  -h, --help               help for routes
  -n, --namespace string   Specified namespace (or default from kubeconfig)
```
